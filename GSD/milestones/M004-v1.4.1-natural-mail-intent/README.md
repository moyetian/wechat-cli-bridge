# M004: v1.4.1 Natural Mail Intent

**Status**: Release Ready

## Goal

在现有 SMTP 邮件通道上增加自然语言纯文本邮件入口，支持：

- 明确收件人的自然语言发信
- `mail.defaultTo` fallback
- bridge 直达发信，不经过 agent

## Non-Goals

- 不在 `M004` 中实现自然语言 HTML 邮件
- 不在 `M004` 中实现自然语言附件邮件
- 不在 `M004` 中引入 IMAP / OAuth / provider SDK
