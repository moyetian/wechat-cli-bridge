# M003: v1.4 Mail Channel

**Status**: Active Milestone, S05 In Progress

## Goal

增加独立于微信的邮件发送能力，支持：

- 纯文本邮件
- HTML 邮件
- 本地文件附件
- 将任务结果发送到指定邮箱

## Why Not In M002

- M002 优先解决当前更核心的微信文件下发问题
- 邮件需要单独的 provider、配置、安全和凭据管理
- 同时做两条外部通道会显著放大验证面和故障面
