## Key Learnings

1. **iLink API 格式**
   ```json
   {
     "msg": {
       "from_user_id": "bot账号",
       "to_user_id": "目标用户",
       "message_type": 2,
       "message_state": 2,
       "context_token": "必须原样返回",
       "item_list": [{"type": 1, "text_item": {"text": "内容"}}]
     }
   }
   ```

2. **认证方式**
   ```
   Authorization: Bearer ${token}
   AuthorizationType: ilink_bot_token
   ```

3. **Windows 命令执行**
   - npm 全局命令需要 shell: true
   - stdin 在 shell 模式下仍可正常工作
